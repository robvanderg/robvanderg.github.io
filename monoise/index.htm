MoNoise Demo

Welcome on the demo page of MoNoise; a modular normalization model. The demo on this page demonstrates a 'translation' of chat texts to more canonical language. More information about how this demo works can be found in the paper and the code.

To try the model, simply choose the language from the dropdown menu, type your text in the input box and press submit. Note that you can tune the aggressiveness of the model using the slider (which will only update after another submit).
